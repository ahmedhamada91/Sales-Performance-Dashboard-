Project Overview
This project involves a comprehensive analysis of sales transactions to monitor business performance and deliver actionable insights. The goal was to create a real-time, interactive dashboard that allows stakeholders to track key metrics across various dimensions like geography, category, and time.
[5:09 ص، 2026/3/5] ahmed hamada: Project Scope & Planning
Following a structured data analysis workflow:  
Target Audience: Designed specifically for Sales Directors to provide high-level and medium-level details.  
KPI Selection: Focused on metrics that matter, including Total Sales ($2.29M), Net Sales ($2.11M), and Total Orders (5009).  
Design Aspect: Planned with a 1920x1080 aspect ratio for clear visual deployment.
[5:10 ص، 2026/3/5] ahmed hamada: Key Features
Data Cleaning: Full transformation of raw sales transactions using Power Query.  
Geographic Analysis: Visualized top-performing states (e.g., California with $457k) and cities (e.g., New York City).  
Category Insights: Comparison of sales across Technology, Furniture, and Office Supplies, with Technology leading in several regions.  
Return Tracking: Analyzed returns to calculate Net Sales accurately.  
Trend Analysis: Yearly breakdown of Sales vs. Quantity to identify growth patterns.
